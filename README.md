# Estrutura de Dados II 💻

Este repositório contém as atividades, exercícios e projetos práticos desenvolvidos para a unidade curricular de **Estrutura de Dados II**. O objetivo principal é implementar, aplicar e compreender o funcionamento de estruturas de dados avançadas, com foco na linguagem **C**.

## 📂 Estrutura do Repositório

O repositório está organizado em diretórios para cada atividade ou avaliação da disciplina. De momento, a organização apresenta-se da seguinte forma:

- 📁 **N1 - Atividade 01/**: [Substituir por uma breve descrição, ex: Implementação de Árvores Binárias de Pesquisa]
- 📁 **N1 - Atividade 02/**: [Substituir por uma breve descrição, ex: Algoritmos de ordenação e Grafos]

*(Esta secção será atualizada à medida que novas atividades e projetos forem submetidos, como trabalhos da N2 ou avaliações finais).*

## 🚀 Tecnologias e Ferramentas

- **Linguagem Principal:** C
- **Compilador Recomendado:** GCC (GNU Compiler Collection)

## ⚙️ Como compilar e executar

Para testar ou correr qualquer um dos exercícios, navega até à pasta da atividade pretendida e utiliza o teu compilador C de preferência.

Exemplo de compilação via terminal:

```bash
# 1. Navegar até à pasta da atividade
cd "N1 - Atividade 01"

# 2. Compilar o ficheiro fonte
gcc nome_do_ficheiro.c -o executavel

# 3. Executar o programa
./executavel   # (Ambientes Linux / macOS)
executavel.exe # (Ambientes Windows)
````
