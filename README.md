# Estruturas de Dados II 💻

Este repositório contém as atividades, exercícios e projetos práticos desenvolvidos para a disciplina de **Estrutura de Dados II**. O objetivo principal é implementar, aplicar e compreender o funcionamento de estruturas de dados avançadas, com foco na linguagem **C**.

## 📂 Estrutura do Repositório

O repositório está organizado em pastas para cada atividade ou avaliação da disciplina. No momento, a organização apresenta-se da seguinte forma:

- 📁 **N1 - Atividade 01/**: [Implementação de uma Tabela Hash com tratamento de colisões].
- 📁 **N1 - Atividade 02/**: [Implementação de Árvores Binárias de Busca].
- 📁 **N2 - Atividade 01/**: [Percurso em Profundidade - Árvore Binária de busca].

*(Esta seção será atualizada à medida que novas atividades e projetos forem adicionados, como trabalhos da N2 ou avaliações finais).*

## 🚀 Tecnologias e Ferramentas

- **Linguagem Principal:** C
- **Compilador Recomendado:** GCC (GNU Compiler Collection) ou similar.

## ⚙️ Como compilar e executar

Para testar ou rodar qualquer um dos exercícios, navegue até a pasta da atividade desejada e utilize o seu compilador C de preferência.

Exemplo de compilação via terminal:

```bash
# 1. Navegar até a pasta da atividade
cd "N1 - Atividade 01"

# 2. Compilar o arquivo fonte
gcc nome_do_arquivo.c -o executavel

# 3. Executar o programa
./executavel   # (Ambientes Linux / macOS)
executavel.exe # (Ambientes Windows)
````
